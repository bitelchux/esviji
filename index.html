Nicolas Hoizey's silly pet project attempting to develop a cross device/browser game with SVG

Be careful, this is work in progress!

Features

  • Vectorial interface, scalled to fit any viewport size without visual loss
  • Multiple interaction options: keyboard, mouse & touch

Still to do

  • Scores persistance
  • Offline
  • Resume previous paused game
  • Package it into pseudo native apps with PhoneGap
  • Open source everything
  • Share feedback on issues encountered during development

Known to work on

  • Most recent desktop browsers: Firefox, Chrome, Safari, Maxthon (this one's for Matthias)
  • Some mobile browsers: iOS Safari, Android stock browser
  • Tell us!

Known to NOT work on

  • Some desktop browsers: Opera (a bug on score handling)
  • Some mobile browsers: Chrome Android
  • Tell us!

Building blocks

  • SVG obviously, but with a little huge help from SVG Web on webkit browsers
  • JavaScript, with a little bit of jQuery
  • Matthias's awesome SVGEventListener polyfil for SVG animate events in webkit (once again)
  • Craig Campbell's mousetrap for keyboard controls
